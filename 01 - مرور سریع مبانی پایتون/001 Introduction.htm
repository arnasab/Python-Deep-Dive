مرور سریع مبانی پایتون

۱. مقدمه

در این بخش، مفاهیم پایه زبان پایتون به‌طور خلاصه مرور می‌شود. این مطالب به شما کمک می‌کند تا برای موضوعات پیشرفته‌تر این دوره آماده شوید. موارد اصلی عبارت‌اند از:

  • سلسله‌مراتب انواع داده‌ها در پایتون
  • قوانین نام‌گذاری
  • نحوه نوشتن خطوط چندگانه
  • اصول شرطی‌ها، توابع، و دستورات کنترلی

۲. سلسله‌مراتب انواع داده‌ها

پایتون دارای سلسله‌مراتب مشخصی از انواع داده‌هاست که برای مدیریت متغیرها و عملیات ضروری است. برخی از انواع کلیدی که در این دوره پوشش داده می‌شوند عبارت‌اند از:

  • اعداد: شامل اعداد صحیح (integers)، اعشاری (floats)، و اعداد مختلط (complex numbers).
  • رشته‌ها (Strings): برای مدیریت متن.
  • ساختارهای داده‌ای: شامل لیست‌ها (Lists)، تاپل‌ها (Tuples)، دیکشنری‌ها (Dictionaries)، و مجموعه‌ها (Sets).

۳. خطوط چندگانه

در پایتون، می‌توان دستورات را به چند خط تقسیم کرد:

  • استفاده از بک‌اسلش (\) برای ادامه دستور.
  • استفاده از ساختارهای توکار (مانند لیست‌ها و تاپل‌ها) برای ایجاد خطوط چندگانه.

۴. قوانین نام‌گذاری

  • نام متغیرها باید با یک حرف یا آندرلاین شروع شود.
  • نباید از کلمات کلیدی پایتون به‌عنوان نام متغیر استفاده شود (مثلاً if، for، while).
  • نام‌ها به حروف بزرگ و کوچک حساس هستند (مثلاً var با Var متفاوت است).

۵. شرطی‌ها (Conditionals)

شرطی‌ها برای اجرای کد بر اساس شرایط خاص استفاده می‌شوند:

  • if: بررسی یک شرط.
  • elif: بررسی شرط دوم در صورت نادرست بودن شرط اول.
  • else: اجرای کد در صورتی که هیچ‌کدام از شرایط برقرار نباشند.
x = 10
if x > 5:
    print("بزرگ‌تر از ۵")
elif x == 5:
    print("برابر با ۵")
else:
    print("کوچک‌تر از ۵")

۶. توابع (Functions)

توابع در پایتون برای سازمان‌دهی کد استفاده می‌شوند:

  • تعریف با استفاده از کلمه کلیدی def.
  • قابلیت بازگشت مقدار با return.
def add(a, b):
    return a + b

result = add(2, 3)
print(result)  # خروجی: ۵

۷. دستورات کنترلی

  • Break: برای خروج از حلقه.
  • Continue: برای رد کردن باقی‌مانده حلقه و ادامه با تکرار بعدی.
  • Try/Except: برای مدیریت خطاها در برنامه.

نتیجه‌گیری

این بخش، یک مرور سریع از مفاهیم پایه‌ای پایتون بود که در این دوره به کار می‌آیند. برای درک بهتر، توصیه می‌شود کدهای نمونه را اجرا کرده و تست کنید.
